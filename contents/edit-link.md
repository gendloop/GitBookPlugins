# edit-link

添加当前文档的编辑图标

## 配置

```json
{
    "plugins": [
        "edit-link@2.0.2"
      ],
    "pluginsConfig": {
        "edit-link": {
            "base": "https://github.com/gendloop/GitBookPlugins/edit/main",
            "label": "Edit"
        }
    }
}
```
